| hilfe | Zeigt eine Liste aller Befehle an. Zusätzlich können Details zu den meisten Gegenständen/Befehlen angezeigt werden. | h, befehle |

| start | Zeigt grundlegende Informationen für den Spieleinstieg. | |
| profil | Zeigt deine aktuellen Statistiken. Nutze 'profil @spieler' um die Statistiken eines anderen Spielers zu sehen. | p |
| inventar | Zeigt dein Inventar. Nutze 'inventar @spieler' um das Inventar eines anderen Spielers zu sehen. | i, inv |
| berufe | Zeigt deinen Fortschritt in den Berufen. Nutze "berufe [beruf]" für genaue Berufserfahrung und Beschreibung. | br |
| bestenliste | Zeigt die Top-Spieler (Server- und globale Bestenlisten verfügbar). | rangliste |
| abklingzeiten | Zeigt deine Abklingzeiten. Nutze 'az @spieler' für die Abklingzeiten eines anderen Spielers. | az |

| quest | Zeigt deine aktuelle Quest oder startet eine neue, wenn möglich. Nutze 'quest ende' zum Abbrechen. | |

| titel | Zeigt verfügbare Titel für dein Profil und wie man sie freischaltet. | |
| erfolge | Zeigt deine Erfolge. Nutze 'erf @spieler' für die Erfolge eines anderen. | erf |

| verlies | Nutze 'verlies @spieler' um mit einem Partner das Verlies des höchsten Gebiets zu betreten und das nächste freizuschalten. | ver |
| arena | Startet eine Arena. Andere Spieler können für Arena-Kekse beitreten. Nutze 'arena @spieler' für verstärkte Belohnungen! | |

| beutekiste | Zeigt verfügbare Beutekisten und deren möglichen Inhalt. | bk |

| verkaufen | Verkaufe Gegenstände aus deinem Inventar. | |
| heilen | Nutze einen Heiltrank um LP zu heilen. | trinken |
| geben | Nutze 'geben @spielername [menge]' um einem anderen Spieler Geld zu geben. | |
| handeln | Handle mit dem EPISCHEN NPC. Handel basiert auf deinem höchsten Gebiet. | |
| epischer-laden | Zeigt den Epischen-Laden für spezielle Gegenstände mit Epischen Münzen. | |
| holzfällen | Gehe Holz hacken, teilt Abklingzeit mit anderen Arbeits-Befehlen. | |
| angeln | Gehe Fischen, teilt Abklingzeit mit anderen Arbeits-Befehlen. | |

| becher | Setze dein Geld ein und versuche ein Quadrat zu finden. | |
| blackjack | Spiele Blackjack mit dem Bot. | bj |
| spielautomat | Setze dein Geld am Spielautomaten ein. | |
| münzwurf | Wirf eine Münze und rate auf welcher Seite sie landet. | mw |
| glücksrad | Drehe am Glücksrad. | |
| lotterie | Nimm an einer Lotterie teil um einen großen Gewinn zu erzielen. | |

| code | Nutze Codes um Belohnungen einzulösen. | |
| event | Zeigt Informationen zu laufenden Events. | events |
| welt | Zeigt aktuelle Verstärkungen für alle Spieler. | |
| verstärkung | Zeigt deine aktuellen Verstärkungen. | |

| bergbau | Grabe nach Münzen und Rubinen, teilt Abklingzeit mit anderen Arbeits-Befehlen. | |
| mehrfachwürfeln | Würfle mit einem anderen Spieler. Nutze 'mehrfachwürfeln @spieler (betrag)'. | |
