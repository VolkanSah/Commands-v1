# Epic RPG Befehle mit deutschen Beschreibungen

## 🎮 Grundbefehle
| Befehl | Beschreibung | Alternative Befehle |
|--------|--------------|-------------------|
| help | Zeigt eine Liste aller Befehle an. Zusätzlich kannst du Details zu Gegenständen/Befehlen sehen. | h, commands |
| rules | Zeigt wichtige Discord-Nutzungsbedingungen, die nicht verletzt werden dürfen. | |
| start | Zeigt grundlegende Informationen für den Spieleinstieg. | |
| profile | Zeigt deine aktuellen Statistiken. Mit '@spieler' siehst du Statistiken anderer. | p |
| inventory | Zeigt dein Inventar. Mit '@spieler' siehst du Inventare anderer. | i, inv |

## 📊 Status & Fortschritt
| Befehl | Beschreibung | Alternative Befehle |
|--------|--------------|-------------------|
| professions | Zeigt deinen Berufe-Fortschritt. Mit '[profession]' siehst du Details zum Beruf. | pr, profession |
| top | Zeigt die Top-Spieler auf Server und global. | leaderboard, ranking |
| cooldowns | Zeigt deine Abklingzeiten. Mit '@spieler' siehst du die von anderen. | cooldown, cd, cds |
| ready | Zeigt nur verfügbare Aktionen. Mit '@player' siehst du die von anderen. | rd |
| title | Zeigt verfügbare Titel und wie man sie freischaltet. | titles |
| achievements | Zeigt deine Erfolge. Mit '@player' siehst du die von anderen. | achievement, ach |
| background | Zeigt verfügbare Profilhintergründe. | bg |
